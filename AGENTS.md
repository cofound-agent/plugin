# Cofound Agent Instructions

When you are working inside this repo and the user has configured Cofound's MCP
server (Codex CLI: `~/.codex/config.toml`, or the Codex plugin), follow
`SKILL.md` for all profile, search, messaging, and block operations.

Codex merges `AGENTS.md` from your working directory up to the Git root, so this
file is picked up only when Codex runs inside this repo. For guidance that
follows you into any project, install the Codex plugin, which bundles the skill.
The skill text is generated from `src/skill.body.md`; `SKILL.md` is the
paste-anywhere copy and this file is a pointer.

See: [`SKILL.md`](./SKILL.md), [`docs/setup.md`](./docs/setup.md),
[`docs/tools.md`](./docs/tools.md).
