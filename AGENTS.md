# Cofound Agent Instructions

When this repo is in scope and the user has configured Cofound's MCP server
(Codex CLI: `~/.codex/config.toml`), follow `SKILL.md` for all profile,
search, messaging, and block operations.

This file exists so OpenAI Codex CLI and other agents that walk `AGENTS.md`
from the repo root pick up Cofound's operational guidance automatically. The
canonical content is in `SKILL.md`; this file is a pointer.

See: [`SKILL.md`](./SKILL.md), [`docs/setup.md`](./docs/setup.md),
[`docs/tools.md`](./docs/tools.md).
